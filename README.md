# ResumeAI 🚀

A full-stack resume analysis platform that helps users understand, improve, and track their resume performance over time.

---

## ✨ Features

- 🔍 Resume analysis with skill detection
- 📊 Score calculation and improvement suggestions
- 📈 Interactive dashboard with analytics
- 👤 User authentication (login/signup)
- 💾 Persistent history using MongoDB
- 🎯 Insights page for strengths and weaknesses

---

## 🧠 What Makes It Different

Unlike basic resume checkers, ResumeAI allows users to track their improvement over time by storing previous analyses and providing structured insights.

---

## 🛠 Tech Stack

Frontend:
- React
- Tailwind CSS
- Chart.js

Backend:
- Node.js
- Express.js

Database:
- MongoDB (Mongoose)
